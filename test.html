UX spent the first quarter of 2019 establishing style standards and guidelines for our applications. Because no such standards previously existed there was a lot of confusion over what the look and feel of our apps should be. So we created a universal stylesheet that we made available to all developers. This cut back on the amount of time they had to sspend doing design work and freed them up to do more development work.

Prototyping is a big part of how we deliver our work. These prototypes are passed along to the dev teams and used as blueprints to build applications. In order to streamline this process, UX has created USPS prototyping templates that exaxty match the styles of the universal stylesheet we provided to the devlopers. This allows us to more rapidly produce high fidelity prototypes, further decreasing production time.

UX has created over 300 icons tailored to the specific needs of our organization. These icons also exist in the form of a font library that we have created for easy use on our web applications

The library that we've built is a robust set of front end tools using a customized version of the tried and true Bootstrap 4 framework. It is scalable, light-weight and all components are mobile ready. If adopted, this tool will reduce development time, reduce development cost, and provide a consistent level of quality accross all of our applications.
